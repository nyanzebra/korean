## 기 나름이다 #card
?begin
### Usage
follows verbs and means "depend on, be up to". Like depends on how one teaches.

Often used to convey it depends on method, effort or skill.
### Form
|     | 기 나름이다  |
| --- | ------- |
| 가다  | 가기 나름이다 |
| 먹다  | 먹기 나름이다 |
### Restrictions
### Examples
* 모든 기계는 사용하기 나름이에요
* 아이가 어떤 사람이 되는가 하는 건 교육하기 나름이죠
