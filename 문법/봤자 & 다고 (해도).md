## 봤자 / 다고 (해도) #card
?
### Usage
Used to mean "even when / even if". 봤자 has a negative connotation and can be self-deprecating.
### Form
#### 봤자

|     | 봤자  |
| --- | --- |
| 읽다  |     |
| 하다  |     |
|     |     |

#### 다고 해도

|     | 다고 해도    |
| --- | -------- |
| 가다  | 간다고 해도   |
| 먹다  | 먹는다고 해도  |
| 바쁘다 | 바쁘다고 해도  |
| 사람  | 사람이라고 해도 |
