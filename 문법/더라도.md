## 더라도 #card
?begin
### Usage
Basically 'even if X, Y' the speaker accepts the content of the preceding clause but it has no effect on second clause
### Form
|       | A/V     | N이다     |
| ----- | ------- | ------- |
| 과거/완료 | -았/었더라도 | 였/이었더라도 |
| 현재    | 더라도     | (이)더라도  |
### Restrictions
| 아/어도                     | 더라도 |
| ------------------------ | --- |
| '-더라도'보다 가정의 느낌이 약할 때, 실 |     |
