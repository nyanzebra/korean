
## 는 텟에 #card
?begin
### Usage
Used to put fault or blame on the previous clause and the result that follows in the next clause.

I can mean 'due to', 'because of', 'as a result of', but the end clause must be some bad result.
### Form
| -   | 탓에     |
| --- | ------ |
| 가다  | 간 탓에   |
| 먹다  | 먹는 탓에  |
| 작다  | 작은 탓에  |
| 바쁘다 | 바쁜 탓에  |
| 동생  | 동생인 탓에 |
### Restrictions
always is an explanation for something bad
### Examples
* 요즘 회사에서 슽레스를 많이 받는 탓에 너무 힘들어요
	* things are hard these days due to getting a lot of stress from work

* 도요들과 어제 술을 많이 맛인 탓에 오늘 아침에 머리가 아팠어요
	* due to drinking a lot with coworkers, today's morning my head hurt
* 더구나, 날씨가 추운 탓에 감기까지 걸려서 지금 컨디션이 안 좋아요
	* moreover, due to the cold weather, I even caught a cold and the condition is not good

|         |                                                                   |
| ------- | ----------------------------------------------------------------- |
| 기 때문에   | neutral reason, can be used for both positive and negative result |
| 는 탓에    | only bad things                                                   |
| 는 바람에   | usually negative, but just means some unexpected occurred         |
| 은/는 덕분에 | exact opposite of 탓에; good outcomes only                          |
?end
