?begin

Show realization, "Oh.."/"I see..."

| 동사  | 군    |

| 하다  | 하는군요 |

| 살다  | 사는군요 |

| 현용사 | 군    |
| --- | ---- |
| 덥다  | 덥군요  |
| 바쁘다 | 바쁘군요 |
