## 다고 해서 다 -은/는 것은 아니에요 #card
?begin
### Usage
This pattern is used when one expresses an opinion that is contradictory to a widely held belief. It means that some principle is not always correct or that it cannot be applied in every single situation.
### Form
This pattern can be used with verbs, adjectives and nouns

### Restrictions
### Examples
1. 좋은 대학교에 간다고 해서 다 성공하는 것은 아니다
2. 문법을 많이 안다고 해서 다 말을 잘하는 것은 아니다
3. 비싸다고 해서 다 좋은 물건은 아니다
4. 선생님이라고 해서 모든 것을 다 아는 것은 아니다
### Notes
The '해서' in '-다고 해서' can be omitted without change in meaning
