## 길 설명하기 표현
### 길을 설명할 때 쓸 수 있는 표현들을 연습해 보세요
1. ()번 출구로 나가다 / 나오다
	1. 193번 출구로 나가세요
	2. 5번 출구로 나오세요
2. -으면 -이 나오다 / 보이다 / 있다
	1. 계속 진행하면 은행이 나와요
	2. (우회전/좌회전)하면 편의점이 있어요
	3. (왼쪽/오른쪽)으로 돌면 편의점이 보인다
3. -을 끼고 -으로 가다 / 오다 / 돌다
	1. 극장을 끼고 오른쪽으로 가세요
	2. 슈퍼를 끼고 왼쪽으로 도세요
	3. 한국어교육원을 끼고 앞쪽으로 오세요
4. -다가 -에서 -으로 가다 / 오다 / 돌다
	1. 가다가 은행에서 오른쪽으로 도세요
	2. 지나가다가 극장에서 왼쪽으로 가세요
	3. 오다가 편의점에서 앞쪽으로 오세요
### 내일 한국어교육원에서 친구를 만나기로 했습니다. 그런데 그 친구는 한국어교육원 위치를 잘 모릅니다. 친구에세 전화로 설명해 주세요
1. 6번 추구 / 나가다/ 나간 방향 / 조금 더 가다 / 세븐 일레븐 / 보이다
	1. 6번 출구로 나가서 나간 방향으로 조금 더 가면 세븐 일레븐이 보여
2. 세븐 일레븐 / 끼다 / 왼쪽 / 돌다
	1. 세븐 일레븐을 끼고 왼쪽으로 돌아
3. 5분 정도 가다 / 서강대학교 정문 / 나오다
	1. 5분 정도 가다가 성강대학교 저문이 나와
4. 정문 / 지나다 / 조금 더 가다 / 왼쪽 / 한국어교육원 / 있다
	1. 정문을 지나서 조금 더 가면 왼쪽에 한국어교육원이 있어
